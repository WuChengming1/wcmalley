studyAjax

  • XHR基本步骤
  • method方法
    1. get 查询数据，url带着查询参数
    2. post 新增数据,数据放在request body不在URL中
    3. put 更新数据,全部更新
    4. patch(打补丁) 更新数据,局部更新
    5. delete 更新数据 全部更新
    6. head 获取数据的信息,不要数据内容
    7. option 测试服务器支持那些方法

接口规范

  • resetful 风格的接口
    1. get /api/books：获取所有书籍数据
    2. get /api/books/007 获取id为007的一本书
    3. get /api/books/？pagenum=1&pagesize = 10 获取第一页十条数据
    4. post /api/books 增加
    5. post /api/books/007 修改id为007的这本书 全量更新 用数据完全覆盖后台数据，只保留id
    6. patct /api/books/007 修改id为007的这本书 局部更新 用数据修改后台数据，如果有新增， 增加
    7. delete /api/books/007 删除id为007的这本书

辅助模拟服务器

  1. 安装 npm install -g json-server
  2. 启动服务器json-server -w books.json
  3. 延时操作 json-server -w -d 3000 books.json
