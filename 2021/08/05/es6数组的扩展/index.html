2. Array.from()将一个类数组转换成真正的数组，有length属性

  • Array,Number,String,Object,Boolean内置函数（它自身的方法属于静态方法）
  1. Array.of()可以把一组参数转为数组
    // 为了避免构造函数 用数字创建数组而引起的歧义
  2. 数组的空位
  3. 判断一个值是不是数组instanceof
    • value instanceof Array

  5. 迭代器方法用于遍历数组keys(),values(),entries()

  10. .slice(切割方法):如果不传进行复制，传一个参数，会从开始索引到结束(包含开始索引)，如果是两个参数，返回起始到结束索引(包含开始索引，不包含结束索引)。

注意：slice(可以传负值)，如果是负值，拿它的length加上这个负值来确定他的位置，如果结束索引小于开始索引返回空数组。

  1. splice():主要是在数组中间插入元素，插入：3个参数分别是，开始元素位置，要删除的元素数量，和插入元素数量任意多个； 删:俩个参数，要删除的索引位和删除的个数，可以多个不限，返回删除的值;改：返回是修改的值
